# EMai

## Emotion Recognizing AI

---

## Information
- This Project is made for Windows 11

---

## Requirements

### Python Libraries

- librosa  
- numpy  
- json  
- os  
- pathlib  
- sounddevice

#### Use `pip install librosa sounddevice numpy` to install packages.

### External Repositories

- Whisper.cpp

#### To install Whisper.cpp separtly run `git clone https://github.com/ggml-org/whisper.cpp`.

---

## To install whole project use:

```bash
git clone https://github.com/dizze00/EMai
cd EMai
git clone https://github.com/ggml-org/whisper.cpp
pip install librosa json os pathlib sounddevice numpy
```
